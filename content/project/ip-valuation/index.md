Submitted to Thailand's Department of Intellectual Property (กรมทรัพย์สินทางปัญญา),
this project developed a rigorous econometric framework for measuring the economic
value of intellectual property assets — including patents, trademarks, and copyrights —
held by Thai businesses. The methodology draws on asset pricing theory and production
function approaches, providing a practical valuation tool for IP-intensive industries
and policymakers assessing the returns to innovation.
